# Finance GPT 

## 📌 Project Overview
Finance GPT is a GenAI-based financial document analysis application designed to automate and enhance financial data processing. The system supports document classification, question answering, summarization, and sensitive data masking.

---

## 🚀 Technologies Used
- Flutter (Frontend)
- FastAPI (Backend)
- NLP Models: Gemini, T5, DistilBERT

---

## 🧠 Key Features
- 📄 Financial Document Classification  
- ❓ Question Answering System  
- 📝 Text Summarization  
- 🔒 Data Masking for sensitive information  
- 📄 Compliance Check

## 🧩 Classification Module
The classification module categorizes financial documents into predefined classes using a fine-tuned DistilBERT model. This helps in automating document organization and improving downstream processing efficiency.
